Mauna Loa Greenhouse Gas Analysis

Analysis of greenhouse gas concentration data recorded above the Mauna Loa Observatory in Hawaii, between the years 2000-2019. 

Techniques used in this project:
- Exploratory Data Analysis
- Analysis of missing data, and imputation via Predictive Mean Matching (PMM)
- Principal Component Analysis (PCA)
- Clustering - K-Means and Agglomerative Hierarchical Clustering.

Key Finding: Three distinct temporal clusters were identified within the data, which corresponded to the year ranges 2000-2005, 2006-2012, 2013-2019, and was driven by structural shifts in greenhouse gas concentrations over time. 

Language used: R
Packages used: tidyverse, mice, factoextra, cluster, VIM, corrplot.
